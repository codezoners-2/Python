string methods


s.lower() / s.upper() # returns the lowercase/uppercase version of the string
s.strip() # returns a string with whitespace removed from the start and end
s.isalpha() / s.isdigit() / s.isspace() # tests if all the string chars are in the various character classes

yet more string methods


s.startswith('other'), s.endswith('other') # tests if the string starts or ends with the given other string
s.find('other') # searches for the given other string, returns the first index where it begins or -1 if not found
s.replace('old', 'new') # returns a string where all occurrences of 'old' have been replaced by 'new'

there are many more

reading a file

one line at a time


file_object = open('myfile.txt', 'r')
line = file_object.readline()
file_object.close()

reading a file

all at once


file_object = open('myfile.txt', 'r')
line = file_object.readlines()
file_object.close()

writing to a file

one string at a time


file_object = open('myfile.txt', 'w')
line = file_object.write("hello world!")
file_object.close()

writing to a file

a list of strings


myStrings = ["the white album", "abbey road", "Let it Be"]
file_object = open('myfile.txt', 'w')
line = file_object.writelines(myStrings)
file_object.close()
